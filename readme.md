# PHP OOP AJAX CRUD

User Management CRUD application built using PHP, AJAX, and MySQL.

## Features
- Add User
- Edit User
- Delete User
- AJAX based updates
- Prepared statements for security

## Tech Stack
- PHP
- MySQL
- JavaScript (AJAX)
- HTML/CSS

## Run Locally
1. Import database.sql
2. Place project inside htdocs
3. Run using XAMPP
